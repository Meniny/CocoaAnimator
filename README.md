# Introduction

## What's this?

`CocoaAnimator` is an `UIView.animationWithDuration` equivalent for `Cocoa` written in `Swift`.

## Requirements

* macOS 10.10+
* Xcode 8+ with Swift 3+

## Installation

#### CocoaPods

```ruby
pod 'CocoaAnimator'
```
